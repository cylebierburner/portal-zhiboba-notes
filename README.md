# portal-zhiboba-notes

## 项目简介

本仓库用于归档和发布一组独立的 HTML 页面。这些页面以静态形式保存，方便浏览、回顾和分享。仓库不绑定任何具体域名或网站，仅作为资料存放与展示的集合。

## 目录说明

```
.
├── README.md          # 本说明文件
├── pages/             # 存放所有 HTML 页面文件
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/            # 页面引用的资源文件（如样式、脚本、图片等）
    ├── style.css
    ├── script.js
    └── ...
```

- `pages/`：所有归档的 HTML 页面均置于此目录下，文件名建议使用有意义的英文或拼音命名。
- `assets/`：存放页面所依赖的外部资源，便于统一管理和引用。

## 页面归档说明

- 每个 HTML 页面均为独立、完整的静态文件，可脱离网络环境直接打开浏览。
- 页面内容可能包含各类信息、笔记或功能演示，不针对任何特定领域或网站。
- 归档遵循“原样保存”原则，不对页面内容进行修改或解释。
- 页面之间的链接关系以相对路径为准，确保在本地或 GitHub Pages 等静态托管环境下正常工作。
